# Bighurra Studios Web App

Questo repository contiene l'applicazione React per il sito.

## Configurazione

1. Clona il repository
2. Entra nella cartella del progetto
3. Installa le dipendenze:

```bash
npm install
```

## Sviluppo

Avvia il server di sviluppo con:

```bash
npm run dev
```

## Build

Genera la versione ottimizzata per la produzione con:

```bash
npm run build
```

## Risorse pubbliche

Per il corretto funzionamento delle pagine di download, inserire i file nella cartella `public/cv` con i seguenti nomi:

- `CV-Fanceschini.pdf`
- `CV-Manente.pdf`
- `Dichiarazioni.pdf`
- `Incarichi.pdf`
- `Programma-Completo.pdf`
- `Volantino-Ufficiale.pdf`
- `Banner-Social-Media.pdf`
- `Locandina-A3.pdf`
- `Kit-Grafico-Completo.zip`

## Note

Il progetto usa Vite, React, Tailwind CSS e React Router.
